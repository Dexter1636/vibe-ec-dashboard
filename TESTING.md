# 快速测试指南

本文档帮助您快速测试运营内容生成工具的完整流程。

## 方式一：自动下载测试图片（推荐）

### 1. 下载测试图片

在项目根目录执行：

```bash
./scripts/download-test-images.sh
```

这个脚本会自动下载 6 张包包相关图片到 `public/test-images/` 文件夹：
- `bag-mens-01.jpg` - 男士商务包
- `bag-mens-02.jpg` - 男士休闲包
- `bag-womens-01.jpg` - 女士手提包
- `bag-womens-02.jpg` - 女士双肩包
- `accessory-01.jpg` - 商务皮带
- `accessory-02.jpg` - 时尚太阳镜

### 2. 启动应用

```bash
npm run dev
```

访问 http://localhost:3000

### 3. 测试完整流程

#### 步骤 1: 加载测试数据

1. 点击顶部导航的「商品任务」标签
2. 在右侧"已添加商品"区域，会看到"暂无商品"的空状态
3. 点击「🚀 加载测试数据」按钮
4. 系统会自动加载 3-6 个测试商品到列表中

#### 步骤 2: 查看已添加的商品

右侧列表会显示所有加载的商品，包括：
- 商品名称和首字母图标
- 类目和品牌信息
- 生成状态（未生成/生成中/已完成）

#### 步骤 3: AI 批量生成内容

1. 点击下方的「✨ 开始生成全部」按钮
2. 观察进度条显示生成进度
3. 等待所有商品生成完成（约 5-10 秒）

#### 步骤 4: 查看和编辑生成结果

生成完成后，每个商品会显示一个结果卡片，包括：

**主图预览**
- 左侧显示商品原图
- 右侧显示生成的主图草稿（带文字叠加）

**标题**
- 显示 AI 生成的商品标题
- 点击「编辑」按钮可以修改标题

**卖点文案**
- 显示 1-2 条卖点
- 点击「编辑」按钮可以修改卖点

**操作按钮**
- 「🔄 重新生成」: 重新生成当前商品的内容
- 「⭐ 收藏模板」: 将当前生成的内容保存到模板库
- 「📥 导出」: 导出当前商品的结果为 JSON 文件

#### 步骤 5: 导出全部结果

1. 点击「📥 导出全部」按钮
2. 下载包含所有商品生成结果的 JSON 文件
3. 文件命名格式：`批量生成结果_时间戳.json`

#### 步骤 6: 收藏模板（可选）

1. 在某个商品的结果卡片中点击「⭐ 收藏模板」
2. 选择要收藏的内容类型（完整模板/仅标题/仅卖点）
3. 切换到「模板库」标签查看收藏的模板

#### 步骤 7: 查看首页概览

1. 点击顶部导航的「首页」标签
2. 查看项目统计数据：
   - 商品总数
   - 已完成生成数量
   - 模板收藏数量
   - 待处理数量
3. 查看任务完成进度条
4. 使用「快速开始」卡片快速导航到其他功能

#### 步骤 8: 配置参数（可选）

1. 点击顶部导航的「设置」标签
2. 配置生成规则：
   - 默认标题长度
   - 卖点数量
   - 是否包含材质信息
3. 配置视觉风格：
   - 主图文字位置
   - 字体大小
   - 文字颜色
4. 点击「保存设置」

## 方式二：使用自己的测试图片

### 1. 准备图片

将您的测试商品图片放到 `public/test-images/` 文件夹中。

### 2. 按规范命名

为了系统自动识别，请按以下规范命名：

```
bag-mens-01.jpg       # 男包图片 1
bag-mens-02.jpg       # 男包图片 2
bag-womens-01.jpg     # 女包图片 1
bag-womens-02.jpg     # 女包图片 2
accessory-01.jpg      # 配饰图片 1
accessory-02.jpg      # 配饰图片 2
```

支持的图片格式：JPG、PNG、WebP

### 3. 加载测试数据

按照「方式一」的步骤 3 执行

## 方式三：手动添加商品

如果您想测试特定的商品信息：

### 1. 点击「添加商品」按钮

在「商品任务」页面，点击左上角的「✨ 添加商品」按钮

### 2. 填写商品信息

- **商品名称**: 例如"男士商务真皮公文包"
- **类目**: 选择或输入（男包/女包/配饰等）
- **品牌**: 例如"VIIBE"
- **材质**: 例如"头层牛皮"
- **尺寸**: 例如"40cm x 30cm x 10cm"
- **颜色**: 例如"黑色"
- **适用人群**: 例如"商务人士"

### 3. 上传商品图片

- 点击上传区域或拖拽图片文件
- 支持 JPG、PNG、WebP 格式
- 建议图片尺寸：800x800px 或以上

### 4. 添加参考图（可选）

- 上传历史爆款截图作为参考
- 或添加参考链接（如商品详情页 URL）

### 5. 保存到素材库（可选）

- 勾选「保存到素材库」
- 方便下次使用相同的商品信息

### 6. 点击「添加商品」

商品会自动添加到列表中

### 7. 重复添加更多商品

重复步骤 1-6 添加更多商品

### 8. 开始生成

按照「方式一」的步骤 3-8 继续测试

## 常见问题

### Q: 点击「加载测试数据」提示"未找到测试图片"

**A**: 请确保：
1. 图片文件已放到 `public/test-images/` 文件夹
2. 文件名符合命名规范（如 `bag-mens-01.jpg`）
3. 图片格式为 JPG、PNG 或 WebP
4. 如果刚添加图片，尝试刷新页面

### Q: 生成的内容是模拟数据，不是真实的 AI 生成？

**A**: 是的，当前版本使用模拟生成来验证流程和交互。接入真实 AI 服务会在后续版本中实现。

### Q: 能否批量导入 Excel 数据？

**A**: 当前版本暂不支持 Excel 导入，后续版本会添加此功能。现在可以使用「加载测试数据」或「手动添加」方式。

### Q: 图片上传后显示不出来？

**A**: 请确保：
1. 图片格式正确（JPG、PNG、WebP）
2. 图片文件没有损坏
3. 图片大小不超过 10MB
